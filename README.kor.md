## 👋 환영합니다!

열정적인 풀스택 개발자로서, 현대적인 웹 애플리케이션과 소프트웨어를 만들고 견고한 인프라를 구축합니다. React/Javascript 또는 Qt/C++를 사용한 프론트엔드와 백엔드 개발부터 Linux 시스템 관리까지, 제가 참여하는 프로젝트에 대한 완벽한 DevOps 비전을 가질 수 있게 해주는 다양한 기술을 보유하고 있습니다.

### 🚀 제가 하는 일

- 고성능 및 확장 가능한 웹 애플리케이션과 소프트웨어 개발
- React를 사용한 현대적인 사용자 인터페이스 제작
- Linux 서버 구성 및 유지보수
- 엔드투엔드 기술 솔루션 설계
- 크로스 플랫폼 개발 (웹, 데스크톱, 모바일)

### 💡 저를 특징짓는 것

- 기술적 다재다능함: 프론트엔드부터 백엔드, DevOps 및 시스템 관리까지
- 오픈소스와 현대 기술에 대한 열정
- 지속적인 개선과 최적화 추구
- 새로운 도구와 프레임워크에 대한 높은 적응력
- 지식 공유에 대한 관심
